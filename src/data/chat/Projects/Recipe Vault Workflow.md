# Recipe Vault Workflow

## Summary
Recipe Vault Workflow explains how to maintain the vault and regenerate the JSON data used by the recipe project.

## Key Points
- Edit recipe notes in `Resources/Recipes` and ingredient notes in `Resources/Ingredients`.
- Run `python export_recipes_to_json.py` from the vault root to refresh the `data/` folder.
- Commit both note changes and regenerated JSON when you update the collection.
