# Machine Settings Guide

## Summary
Machine Settings Guide explains how to adjust common gym machines so the movement targets the intended muscles, feels more stable, and avoids obvious setup errors.

## Key Points
- On [[Leg Press]], set the seat so the knees can bend deeply without the hips rolling off the pad. This keeps the quads and glutes working while reducing low-back compensation.
- On [[Chest Press]], raise or lower the seat until the handles start around mid-chest. This keeps the chest, shoulders, and triceps in a better pressing line.
- On [[Shoulder Press]], set the seat so the handles begin near ear level. This helps the shoulders press vertically without turning the movement into an awkward chest press.
- On [[Lat Pulldown]], lock the thigh pad snugly over the legs. This stops the body lifting off the seat and lets the lats do more of the pulling.
- On [[Seated Row]], set the chest pad or seat so the shoulders can reach forward without collapsing the torso. This gives the upper back room to move through the full pull.
- On [[Leg Extension]], line the machine pivot with the knee joint and place the pad above the ankle. This makes the quads do the work instead of stressing the joint through a poor lever.
- On [[Leg Curl]], align the knee with the machine pivot and secure the lower-leg pad comfortably. This improves hamstring tension and reduces hip compensation.
- On [[Abdominal Crunch]], set the pads so the torso can flex without the neck doing the work. This shifts the effort back to the trunk.
- In general, seat height changes the angle of push or pull, pad position changes leverage, and handle position changes which muscles work hardest.
- If a setting forces shrugging, joint pinching, or loss of contact with the pad, the machine usually needs adjustment before the set starts.
