# Machine Setup Help

## Summary
Machine Setup Help defines how the chatbot should answer gym machine adjustment and setup questions using the equipment and safety notes in the vault.

## Key Points
- The chatbot should answer questions about seat height, pad position, handle position, starting posture, and what each setup change affects.
- The main vault sources for setup answers are [[Machine Settings Guide]], [[Gym Equipment Safety Guide]], and [[Gym Equipment Hub]].
- Good machine setup answers should say what to adjust, what muscle or movement the setting changes, and what common setup mistake to avoid.
- When the question is about a specific machine, the chatbot should connect the answer to the machine note and any related machine hub.
- The chatbot should avoid sounding certain about a machine model it cannot see if the actual gym setup may differ from the vault example.
