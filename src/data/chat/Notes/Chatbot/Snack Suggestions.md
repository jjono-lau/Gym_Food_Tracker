# Snack Suggestions

## Summary
Snack Suggestions defines how the chatbot should answer snack questions using practical, health-aware options from the vault.

## Key Points
- The chatbot should answer snack questions with 2 to 4 realistic options instead of broad lists.
- Good snack answers should prefer higher-fiber, lower-sugar, and more filling choices when the user asks about PCOS or high triglycerides.
- Useful snack patterns include fruit with protein, yogurt-based options, nuts in moderate portions, and simple minimally processed items.
- When the vault does not yet have a dedicated snack recipe, the chatbot should suggest ingredient-based snack combinations instead of inventing full recipes.
- Snack answers should connect back to [[Meal Suggestions]], [[Ingredient Swaps]], and the relevant health guidance note when needed.
