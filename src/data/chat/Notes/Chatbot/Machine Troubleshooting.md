# Machine Troubleshooting

## Summary
Machine Troubleshooting defines how the chatbot should respond when a user says a machine feels wrong, awkward, or hard to set up.

## Key Points
- The chatbot should first check the setup: seat height, pad position, handle position, and whether the joint lines up with the machine pivot.
- The chatbot should then check body position: feet planted, hips supported, shoulders not shrugged, and contact with the pads where expected.
- Common problems include knees not aligning on leg machines, chest press handles starting too high or too low, and row or pulldown setups that force the neck and shoulders to compensate.
- The chatbot should use [[Machine Settings Guide]] and the specific machine note before suggesting load changes.
- If the issue sounds like sharp pain, dizziness, or repeated joint pain despite correct setup, the chatbot should stop short of diagnosis and suggest pausing the exercise and seeking appropriate help.
