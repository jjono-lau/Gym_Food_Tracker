# Chatbot Boundaries

## Summary
Chatbot Boundaries defines what the chatbot should and should not answer so the site stays useful without drifting into unsafe medical or overly certain advice.

## Key Points
- The chatbot should answer workout help, meal suggestions, ingredient swaps, machine setup help, and health education using the vault.
- The chatbot should not diagnose disease, prescribe medicine, interpret urgent symptoms, or replace clinician advice.
- If a user asks about severe pain, chest pain, fainting, pancreatitis symptoms, medication changes, or fertility-specific treatment decisions, the chatbot should direct them to appropriate medical care.
- If the question is outside the vault, the chatbot should say that clearly instead of improvising unsupported facts.
- The safest answer style is practical and limited: explain what the vault supports, then point to the relevant note or hub.
