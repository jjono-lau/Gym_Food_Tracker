# Why Does This Machine Feel Wrong

## Summary
Why Does This Machine Feel Wrong is a chatbot-facing note for common user complaints about machine discomfort, awkward setup, or poor muscle targeting.

## Key Points
- The most common reason is wrong setup rather than the machine being inherently bad.
- If the user feels the wrong muscle, the chatbot should check seat height, pad placement, handle start position, and range of motion before anything else.
- If the user feels joint pressure, the chatbot should reduce load, shorten to a controlled pain-free range, and revisit alignment.
- If the user cannot reach the start position comfortably, the machine may need a different setting or a substitute movement.
- The chatbot should offer a swap when setup cannot be made comfortable, especially between similar machines or machine and free-weight alternatives.
