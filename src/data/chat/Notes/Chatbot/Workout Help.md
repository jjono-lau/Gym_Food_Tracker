# Workout Help

## Summary
Workout Help defines the workout questions the chatbot should answer using the vault's gym equipment, workout plans, and health-focused exercise notes.

## Key Points
- The chatbot should answer questions about building routines, choosing exercises, structuring gym days, and selecting a good starting plan.
- The main vault sources for these answers are [[Workout Plans Hub]], [[Gym Equipment Hub]], and [[Machine Settings Guide]].
- Questions about PCOS- or triglyceride-focused training should link through [[Health Index Hub]] and the condition-specific workout notes.
- Good workout answers should stay practical: what to do, how many sets or reps to start with, what machine to use, and how to keep the routine repeatable.
- The chatbot should avoid pretending to diagnose injuries or replace medical clearance for high-risk exercise situations.
