# Smith Machine

## Summary
A Smith Machine guides a fixed bar path for squats, presses, lunges, rows, and other barbell-style movements.

## Key Points
- Do: set the safeties before the set and rehearse the path with the empty bar because the bar does not move freely.
- Do: position your feet to match the fixed rail path instead of forcing your natural free-weight pattern onto the machine.
- Do not: assume the machine makes every movement safe; poor setup still puts you in bad positions.
- Do not: twist the wrists unevenly to rerack the bar under fatigue.
