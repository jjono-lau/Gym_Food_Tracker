# Freedom Rack

## Summary
A Freedom Rack is a multi-purpose rack station designed for barbell training, pull-ups, and accessory setup work.

## Key Points
- Do: use it for compound lifts when you need a stable station with adjustable supports.
- Do: set every attachment and support point before the bar is loaded.
- Do not: use features you have not checked for correct locking or height.
- Do not: overload one side of the station while the other side remains empty if the design depends on balanced loading.
