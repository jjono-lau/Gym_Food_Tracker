# Leg Curl

## Summary
A Leg Curl isolates the hamstrings through knee flexion using a seated or lying pad system.

## Key Points
- Do: line up the knee with the machine pivot and set the pad so it contacts the lower leg securely.
- Do: keep your hips pinned to the pad so the hamstrings do the work.
- Do not: arch or lift your hips to help the weight move.
- Do not: rush the lowering phase because hamstrings respond well to controlled eccentrics.
