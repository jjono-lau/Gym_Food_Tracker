# Bilateral Arm Curl

## Summary
A Bilateral Arm Curl machine trains both arms together through a stable curl pattern with guided resistance.

## Key Points
- Do: keep both wrists neutral and apply even force through each side of the machine.
- Do: sit or stand in the machine exactly as intended so the axis and pads line up correctly.
- Do not: twist one shoulder forward to make up for a weaker side.
- Do not: shorten the eccentric just to start the next rep faster.
