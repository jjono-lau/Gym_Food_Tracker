# Adjustable Bench

## Summary
An Adjustable Bench changes angle so you can train presses, rows, curls, and seated work at multiple positions.

## Key Points
- Do: lock the back pad and seat before you sit down and test the angle with bodyweight first.
- Do: keep your head, upper back, and hips supported when the exercise calls for bench contact.
- Do not: set the bench on unstable flooring or where it blocks walkways and rack exits.
- Do not: use a partially locked bench because the back pad can shift under load.
