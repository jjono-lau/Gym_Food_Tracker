# Pull Over

## Summary
A Pull Over machine trains shoulder extension and the lats through an arcing upper body pulling motion.

## Key Points
- Do: set the seat and arm position so you can start with a strong stretch without shoulder strain.
- Do: pull through the elbows and upper arm path rather than curling the wrists and forearms.
- Do not: force the top position if your shoulders do not tolerate it comfortably.
- Do not: rush the return, because the long lever increases joint stress when uncontrolled.
