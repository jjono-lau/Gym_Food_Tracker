# Biceps Curl

## Summary
A Biceps Curl machine isolates elbow flexion with a fixed arm path and stable upper arm support.

## Key Points
- Do: match the machine pivot to your elbow line and keep your upper arm planted as you curl.
- Do: use smooth reps and control the lowering phase so the biceps stay loaded.
- Do not: roll the shoulders forward to chase a shorter top range.
- Do not: let the weight stack crash at the bottom of each repetition.
