# Back Extension

## Summary
A Back Extension machine trains trunk extension and posterior chain strength through a supported hip hinge pattern.

## Key Points
- Do: set the machine so the hips move freely while the torso stays supported and aligned.
- Do: finish each rep in neutral, with the glutes and low back sharing the load.
- Do not: jam into lumbar hyperextension to make the top look bigger.
- Do not: use momentum to swing through the bottom and top positions.
