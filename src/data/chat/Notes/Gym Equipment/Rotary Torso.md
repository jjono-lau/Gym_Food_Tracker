# Rotary Torso

## Summary
A Rotary Torso machine trains the obliques and trunk rotators through a fixed rotational path.

## Key Points
- Do: set the seat and chest position so the machine rotates through your midsection rather than your knees.
- Do: use a moderate range and steady tempo because the spine usually prefers controlled rotation.
- Do not: force the deepest possible twist on every repetition.
- Do not: bounce off the end range or use excessive load for a torso rotation pattern.
