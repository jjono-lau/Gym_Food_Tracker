# Dual Adjustable Pulley

## Summary
A Dual Adjustable Pulley uses adjustable cable arms for rows, presses, raises, pull-down style work, and many unilateral exercises.

## Key Points
- Do: set both pulleys to the same height when the exercise needs symmetry and step into a stable stance before pulling.
- Do: control the eccentric so the handles return quietly instead of snapping the stack.
- Do not: stand too close to the weight stack if the cable will pull you out of position at the start.
- Do not: let the handles fly back because that stresses the cable system and your joints.
