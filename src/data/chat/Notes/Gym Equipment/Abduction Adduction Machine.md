# Abduction Adduction Machine

## Summary
An Abduction Adduction Machine trains the hips by moving the legs outward for abduction or inward for adduction.

## Key Points
- Do: sit upright, set the start range comfortably, and move from the hips rather than rocking the torso.
- Do: use a controlled tempo because these muscles respond better to tension than momentum.
- Do not: slam into the end stops on either side of the movement.
- Do not: lean sideways to create fake range of motion.
